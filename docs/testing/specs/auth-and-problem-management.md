# 鉴权与题目管理测试规范

## 鉴权场景
- 无 token 访问受保护接口：`401 / 40100`
- 非法 token 访问受保护接口：`401 / 40100`
- 普通用户访问管理员接口：`403 / 40300`

## 题目管理场景
- 管理员创建题目成功：`200 / code=0`
- 管理员编辑题目成功：`200 / code=0`
- 删除被比赛/题单关联的题目：`409 / 40900`
- 删除无关联题目成功：`200 / code=0`

## 断言要求
- 写操作必须包含数据层断言（通过 Mapper 查询结果）。
- 冲突场景必须断言冲突错误码 `40900`。
- 权限场景必须区分未登录（40100）与已登录但无权限（40300）。
